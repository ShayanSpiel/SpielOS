# /analyze

**State machine:** Content — ANALYZING only
**Template:** AGENTS.md

## Usage
```
/analyze            # Analyze all posted content
/analyze 30d        # Analyze last N days
```

## Action
- Aggregate posted drafts by platform, pattern, engagement_ask
- Compare rejected patterns against posted patterns
- If clear failure pattern emerges: update style guide
- If engagement data available: note which patterns perform
- Log analysis results
