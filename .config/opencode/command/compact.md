# /compact

**State machine:** Wiki — special synthesis state
**Template:** AGENTS.md

## Usage
```
/compact [topic]    # Consolidate related concepts into summary
```

## Action
- Search for all pages tagged with or related to topic
- Read each page, extract key claims and links
- Create a new summary page synthesizing the information
- Add [[wikilinks]] back to source pages
- Proceed to INDEXING → VALIDATING
