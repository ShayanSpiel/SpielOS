# Researcher Archive Note

Former job:
- capture raw sessions
- synthesize the current conversation
- classify into a start point for the compiler

Why it was removed:
- it created a blocking pre-step
- it was too easy to produce fake certainty from empty sessions
- the compiler already needs the source, so the extra role added ceremony

Restore it when:
- session capture becomes a real product surface again
- there is enough signal to justify separate source synthesis
