# Pipeline

SpielOS is a lean role-based marketing team.

```text
Director -> Strategist -> Writer -> Editor -> Publisher -> Director
```

## Roles

| Step | Role | Output |
|---|---|---|
| 1 | Director | source accepted and `content/current.md` created |
| 2 | Strategist | reader, pain, point, proof, angle, formats |
| 3 | Writer | draft files in `content/drafts/` |
| 4 | Editor | ready files in `content/ready/` |
| 5 | Publisher | posted, held, or rejected files |

## Rule

No source, no post.

If `/post` has no concrete source, Director stops and asks for one.
