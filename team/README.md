# Team

The live product has five roles:

| Role | File | Job |
|---|---|---|
| Director | `team/director.md` | route source and delegate |
| Strategist | `team/strategist.md` | create reader, pain, point, proof, angle |
| Writer | `team/writer.md` | write drafts |
| Editor | `team/editor.md` | make drafts ready |
| Publisher | `team/publisher.md` | publish, hold, or reject |

No nested subagents. No session-capture state. No designer or analyst in the live path.
