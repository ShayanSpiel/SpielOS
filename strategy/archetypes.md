# Session Archetypes

| # | Archetype | Description |
|---|---|---|
| S1 | System Build | Building a system, architecture, or workflow |
| S2 | Ship | Shipping a feature, product, or release |
| S3 | Decision | Choosing X over Y, documented trade-offs |
| S4 | Lesson | Something learned, abstracted into insight |
| S5 | Failure | Something broke, went wrong, got fixed |
| S6 | Client Work | Work done for/with someone else |
| S7 | Research | Learning, reading, analyzing |
| S8 | Tooling | Building tools, scripts, automations |
| S9 | Strategy | Planning, positioning, thinking |
| S10 | Meta | Working on the system itself |
