Install SpielOS

Run this in any project directory on your Mac or Linux machine. The installer puts the vault at your current directory, starts the setup wizard, and syncs the 5 role agents + 3 skills to all your IDEs.

curl -fsSL https://spielos.xyz/spielos | bash

Requires python3 and git on PATH. Install: brew install python git (Mac) or apt install python3 git (Linux).

← Back to spielos.xyz/SpielOS

What just happened

1. Vault installed at $PWD

The installer cloned the SpielOS repo into your current directory. This is your vault — the project where your drafts, strategy files, and brand tokens live. Any IDE opened in this directory sees your project.

2. Wizard wrote 8 files

The setup wizard at http://localhost:7331 walked you through 6 steps: brand, audience, offer, voice, examples, connect. On Finish it wrote your system/brand.*, 4 strategy/*.md files, and the .env for API tokens.

3. Adapters installed in 4 IDEs

The shim ~/.local/bin/spiel is on your PATH. Global config ~/.config/spielos/config points at this vault. The 5 role agents (Director, Strategist, Writer, Editor, Publisher) and 3 skills (format, publish, voice) are now in ~/.config/opencode/, ~/.claude/, ~/.cursor/, and ~/.codex/.

Now

Open this project in Cursor, opencode, Claude Code, or Codex. The chat box now has a /post slash command. Try:

/post "Just shipped v2"

The Director subagent will:

  1. Write content/current.md with your source
  2. Strategist compiles reader / pain / point / proof / angle
  3. Writer asks you which platforms (X, LinkedIn, blog)
  4. Drafts land in content/drafts/
  5. Editor runs 4 mechanical gates, moves passing drafts to content/ready/
  6. Publisher asks publish / hold / reject, dispatches approved ones

Updating later

When a new version ships:

spiel update

Pulls the latest tools and wizard, syncs your IDEs, preserves your team/, skills/, strategy/, content/, .env, and system/brand.*.
